For more information about the kits installed in this folder, please visit
the Spicy Pixel website at http://spicypixel.com.

Offline documentation is provided in the form of a CHM file. Windows can 
read CHM files natively when not stored on a network location. 

There are apps available in the Mac app store that can display CHM files 
as well:

http://itunes.apple.com/us/app/chm-viewer/id417572769?mt=12
http://itunes.apple.com/us/app/chm-view/id449814591?mt=12
http://chmox.sourceforge.net/
